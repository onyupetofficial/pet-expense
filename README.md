# 온유펫 - 반려동물 가계부

반려동물 양육에 드는 지출을 체계적으로 기록하고 분석하는 가계부 앱입니다.

## 주요 기능

- 사료, 간식, 의료, 미용 등 **10가지 카테고리**로 지출 분류
- 영수증/물품 **사진 다중 첨부**
- **월별 바 차트 + 카테고리별 파이 차트**로 시각적 분석
- 전월 대비 지출 **변동률** 한눈에 확인
- JSON/CSV **데이터 백업 및 복원**
- 클라우드 동기화로 기기 변경 시에도 데이터 보존
